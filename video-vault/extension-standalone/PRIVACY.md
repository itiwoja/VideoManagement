# プライバシーポリシー — Video Vault Chrome 拡張機能

最終更新: 2026-05-06

## TL;DR

**この拡張機能は、いかなる個人情報・利用データも収集・送信しません。**
すべてのデータはあなたのブラウザ内 (`chrome.storage.local`) に保存され、開発元 (村上一輝) を含む第三者には一切送信されません。

## 取得する情報

なし。

具体的には:
- ユーザー登録は不要 (アカウントなし)
- アクセス解析ツールは導入していない
- エラーレポート機能なし
- ネットワーク経由でのデータ送信なし

## 保存される情報 (ローカルのみ)

以下の情報は **あなたの Chrome のローカルストレージ** (`chrome.storage.local`) にのみ保存されます。
開発元・第三者は **アクセスできません**。

- 保存した動画の URL
- ページタイトル
- サムネイル URL (YouTube 等から自動推定)
- あなたが入力したタグ・評価・メモ
- 視聴回数 / 最終視聴日 (ローカルでカウント)

## 拡張機能の権限について

この拡張機能は以下の権限を要求しますが、いずれも **データ送信のためではありません**:

| 権限 | 用途 |
|------|------|
| `contextMenus` | 右クリックメニューに「Vault に保存」を表示 |
| `activeTab` | 現在のタブの URL とタイトルを取得 (保存ボタン押下時のみ) |
| `storage` | 動画リストをブラウザ内に保存 |
| `notifications` | 保存完了の通知を表示 |
| `tabs` | 「Vault を開く」で新しいタブを開く |
| `host_permissions: <all_urls>` | アプリ内再生で iframe / `<video>` タグから外部サイトを表示 (これは表示のみで、データ送信ではない) |

## データの移転と削除

- **エクスポート**: Vault 画面の「↓ エクスポート」で JSON 形式でダウンロード可能
- **インポート**: 別端末で同 JSON を読み込めば移行可能
- **削除**: 拡張機能をアンインストール、または `chrome://settings/cookies` から `chrome-extension://...` のストレージを削除

## サードパーティサービス

なし。

ただし以下の点に注意:

- **動画の埋め込み再生**: アプリ内で動画を再生する際、YouTube / TikTok 等の元サービスから iframe / 動画ファイルを直接取得します。これは標準的な埋め込み再生であり、それらのサービスにアクセスログが残る可能性があります (これは Chrome ブラウザで動画を視聴するのと同じです)。
- **YouTube サムネ自動推定**: 保存時に YouTube URL を検出した場合、`https://i.ytimg.com/vi/<videoId>/hqdefault.jpg` のサムネ URL を保存します。サムネ画像の表示時に YouTube から画像を取得します。
