# Video Vault — Chrome 拡張機能 (スタンドアロン版)

> **気になる動画 URL を 1 タップで保存して、後でタグ・評価・メモ付きで管理できるブックマーク拡張機能。**

サーバー不要、データはあなたのブラウザに保存されます (chrome.storage.local)。

## 主な機能

- **🚀 ワンクリック保存** — ツールバーアイコン / 右クリック / Ctrl+Shift+V
- **🎬 アプリ内再生** — YouTube / TikTok / ニコニコ動画 / Vimeo / 直リンク .mp4
- **🏷️ タグ + 評価 (★1-5) + メモ** で整理
- **🔍 検索 + フィルタ** (サイト / タグ / 評価)
- **📤 エクスポート / インポート (JSON)** — 端末間移行・バックアップ
- **🌙 ダークモード前提のデザイン**
- **📱 レスポンシブ** — スマホブラウザでも使える

### 開発版を手動インストール (今すぐ試したい人向け)

1. このディレクトリをローカルにクローン or ダウンロード
2. Chrome (or Edge / Brave) で `chrome://extensions/` を開く
3. 右上「**デベロッパーモード**」を ON
4. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
5. `extension-standalone/` フォルダを選択
6. ツールバーに 📚 アイコンが表示されたら完了

## 使い方

### 動画を保存する

3 通り:

- **ツールバーの拡張機能アイコンをクリック** → ポップアップから「📥 このページを Vault に保存」
- **ページ / リンク / 動画の上で右クリック** → 「Vault に保存」
- **キーボード**: `Ctrl + Shift + V` (Mac は `Cmd + Shift + V`)

### Vault を開く

- ツールバーの拡張機能アイコン → ポップアップ右上の「📚 Vault」をクリック
- または `chrome-extension://<拡張ID>/vault.html` を直接開く

### 動画を再生する

カードのサムネ or タイトルをクリック:
- YouTube / TikTok / ニコニコ動画 / Vimeo → アプリ内 iframe で再生
- .mp4 / .webm 直リンク → アプリ内 `<video>` タグで再生
- それ以外 → 「元サイトで開く」ボタンで新規タブ

### バックアップ・移行

- **エクスポート**: ヘッダー右の「↓ エクスポート」で JSON ダウンロード
- **インポート**: 「↑ インポート」で別端末からのバックアップを読み込む (重複 URL は自動スキップ)

## データの保存場所

すべてのデータは **あなたの Chrome ローカルストレージ (chrome.storage.local)** に保存されます。

- ✅ ネット送信は **一切ありません** (アプリ内再生時の元サイト fetch を除く)
- ✅ サーバー不要、登録不要、月額無料
- ✅ ブラウザ内で完結
- ⚠️ 端末を変える / Chrome をアンインストールするとデータが消える → **定期的に JSON エクスポートでバックアップを推奨**

## 制限

- chrome.storage.local の容量上限は 10 MB → URL とメタデータのみなので **数千件は保存可能**
- 動画ファイル本体はダウンロードしない (URL のみ管理)
- アプリ内再生は **iframe / 直リンク .mp4 / .webm** のみ。それ以外のサイトは元サイトで開く
- 1 端末完結 (クラウド同期なし、JSON エクスポート/インポートで対応)

## トラブルシューティング

### 「保存できませんでした」と出る
- `chrome://` や `about://` のページ、Chrome 内部ページは保存できない仕様です
- ブラウザを再起動して再度試してください

### サムネが表示されない
- YouTube は自動推定でサムネを取得します。それ以外のサイトは サムネ無し で保存されます
- 編集画面でサムネ URL を手動指定可能 (将来追加予定)

### Cookie / Referer 必要なサイトで再生できない
- 一部のサイト (動画配信サイト等) は埋め込み再生不可。「元サイトで開く」ボタンを使ってください
